# Lista de verificación de producción de video

Este archivo es interno para el equipo. El estudiante no necesita leerlo para completar el curso.

## Antes de grabar

- [ ] Definir objetivo de la lección en una frase.
- [ ] Abrir el editor online con el código inicial.
- [ ] Aumentar zoom del navegador para que el código sea legible.
- [ ] Probar micrófono.
- [ ] Cerrar notificaciones.
- [ ] Preparar una pausa de práctica.

## Durante la grabación

- [ ] Explicar primero el problema y luego el código.
- [ ] Leer el código de arriba hacia abajo.
- [ ] Evitar jerga innecesaria.
- [ ] Usar ejemplos cercanos al público objetivo.
- [ ] Mostrar el resultado en consola.
- [ ] Hacer una pausa para que el estudiante practique.

## Después de grabar

- [ ] Cortar silencios largos.
- [ ] Revisar que el audio sea claro.
- [ ] Revisar que el código se pueda leer.
- [ ] Subir como video no listado.
- [ ] Pegar el enlace en `course-plan.md`.
